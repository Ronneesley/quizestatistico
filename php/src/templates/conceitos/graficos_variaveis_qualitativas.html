Gráfico de Barras

Um gráfico de barras é uma forma de resumir um conjunto de dados categóricos. Ele mostra os dados utilizando um número de barras de mesma largura, cada uma delas representando uma categoria particular. A altura de cada barra é proporcional a uma agregaçõo específica (por ex., a soma dos valores na categoria que representa). As categorias podem ser algo como um grupo de idade ou uma localização geográfica.

Razão pela qual os gráficos de barras são tão populares é que eles são simples, mas versáteis. As barras podem ser alinhadas verticalmente, como no exemplo acima, ou horizontalmente.

Exemplo:

Uma tabela de dados contém os números de venda para um número de diferentes frutas e legumes. O gráfico de barras pode mostrar a soma total das vendas de diferentes anos.

Gráfico de Colunas

Um gráfico de colunas é um método de exibição de dados com categorias representadas por um retângulo, às vezes chamado de gráficos de barras verticais. Eles permitem comparações fáceis entre vários itens e análise de tendências. Em geral, estatísticas e figuras são difíceis de entender quando apresentadas em tabelas ou em formato escrito. Os gráficos de colunas tornam as coisas fáceis, diretas e compreensíveis.





Gráfico de linhas

Gráficos de linhas mostram tendências ou alterações ao longo do tempo exibindo uma série de pontos de dados conectados por segmentos de linha reta. Você pode exibir um único campo de dados como várias linhas com base em diferentes categorias por meio de Cor por campo.

Os gráficos de linhas costumam ser usados para mostrar dados contínuos, ou seja, que não podem ser contados em números inteiros, como o número de produtos vendidos. Em vez disso, esses dados estão sempre variando e podem ter qualquer valor imaginável, incluindo frações ou pontos decimais, como tempo, temperatura, medições, custo, etc.





Gráfico de setores

O gráfico de setores, também conhecido por gráfico de pizza ou circular, é uma representação de dados de categorias diferentes na forma de fatias ou setores circulares.

É uma ferramenta estatística utilizada para analisar e expressar uma comparação entre as quantidades de dados obtidos para categorias diferentes em uma pesquisa ou contagem.

As fatias ou setores ocupam áreas diferentes, conforme a proporção entre as quantidades de dados de cada categoria. Esta proporção é representada na forma de porcentagem, onde setores com maior porcentagem, aparecem com áreas maiores.
